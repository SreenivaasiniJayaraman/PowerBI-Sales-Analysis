# PowerBI-Sales-Analysis
Sales Performance Analysis using Power BI
Sales Performance Analysis using Power BI

Project Overview

This project analyzes sales data using Microsoft Power BI to understand business performance.
The dashboard provides insights into total sales, category-wise performance, regional sales distribution, and sales trends over time.

Tools Used

- Microsoft Power BI
- CSV Dataset
- GitHub for project hosting

Dataset

The dataset contains information about:

- Product Category
- Region
- Order Date
- Sales Amount
- Customer Details

Dashboard Visualizations

The Power BI dashboard includes the following visualizations:

1. Total Sales Card – Displays the total sales value.
2. Bar Chart (Sales by Category) – Shows sales comparison across product categories.
3. Pie Chart (Sales by Region) – Displays regional sales distribution.
4. Line Chart (Sales Trend) – Shows sales growth over time.

Key Insights

- Total sales recorded in the dataset is 2.26 Million.
- The Technology category has the highest sales.
- The West region contributes the highest revenue.
- Sales show fluctuations across different months and years.

Project Files

- "Sales_Dashboard.pbix" – Power BI dashboard file
- "train.csv" – Dataset used for analysis

Conclusion

This project demonstrates how Power BI can be used to analyze business data and create interactive dashboards that help in understanding sales performance and identifying trends.
